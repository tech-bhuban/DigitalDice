
# Digital Dice Roller

## What it does
A fun dice roller app with visual dice faces.

## How it works
1. Shows a dice face
2. Click "Roll Dice" button
3. Random number 1-6 appears
4. Shows actual dice symbols

## Dice Faces
- ⚀ = 1
- ⚁ = 2  
- ⚂ = 3
- ⚃ = 4
- ⚄ = 5
- ⚅ = 6

## Features
- Real dice symbols
- Big, easy to see
- Fun to play with
